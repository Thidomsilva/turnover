# **App Name**: Exit Insights

## Core Features:

- Exit Data Entry: Simple and responsive form for entering employee exit data.
- Data Import: Import historical exit data from Excel spreadsheets.
- Dashboard: Dashboard displaying key metrics such as total exits, types of exits, and reasons for exits, per month and year.
- Data Filtering: Filters for data analysis based on period, department, exit type, and manager.
- Report Export: Exportable reports in PDF and Excel formats for sharing insights.
- Data Management: Interface for editing existing exit records and data maintenance.
- AI Insights Generator: AI-powered tool to generate insights about potential issues, by considering all exit data (turnover rates, top reasons for leaving, department issues) and flag important observations for HR and management.  The tool will decide if some reason appears multiple times, is unexpectedly high, or shows correlation to a manager's area of responsiblity.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and stability, vital for HR data analysis.
- Background color: Light gray (#ECEFF1) for a clean, modern, and analytical workspace.
- Accent color: Soft purple (#9575CD) for highlighting key metrics and interactive elements.
- Body and headline font: 'Inter' (sans-serif) for clear readability and a modern feel. Suitable for both data tables and dashboard titles.
- Simple, line-style icons to represent different data types and actions in the dashboard and data entry forms.
- Clean and structured layout with a focus on data visualization, using cards and tables for organized presentation.
- Subtle transitions and animations to enhance user interaction, such as loading states and filter updates.
